Compilação:
No terminal acesse a pasta Projeto Bares e insira o seguinte comando.
g++ src/main.cpp -o bin/prog

Execução:
Após a compilação ainda na pasta /Projeto Bares insira o seguinte comando.
Caso esteja no Linux: bin/prog <arquivo de entrada> [arquivo de saida]
Caso esteja no Windows: bin/prog.exe <arquivo de entrada> [arquivo de saida]
Limitações: Os arquivos de entrada e saída devem estar obrigatoriamente na pasta /data do Projeto Bares.

Tratamento de error:
O programa trata os seguintes erros.
- Constante numérica inválida.
- Falta de operando.
- Operando inválido.
- Divisão por zero.
